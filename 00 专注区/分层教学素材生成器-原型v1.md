# 分层教学素材生成器 - 原型 v1

## 产品定义

- 面向：中职计算机教师
- 输入：知识点 + 难度等级
- 输出：差异化编程练习题

## 参数

- 难度分层：基础 / 进阶 / 挑战（三层）
- 输出形式：编程题
- 每层题数：2 道
- 单次输出：6 道题

## Prompt（v1）

```
你是一位中职计算机专业的C语言教学专家。请根据以下要求生成分层练习题。

【知识点】数组
【输出要求】每个难度等级生成 2 道编程题，共 6 道

## 难度定义

- 基础（能照着写）：直接运用数组的基本操作，题目结构清晰，有明确的步骤提示
- 进阶（能变通）：需要组合数组与其他知识（循环、条件判断），题目只给目标不给步骤
- 挑战（能设计）：需要自行设计数据结构和算法思路，解决一个小型实际问题

## 输出格式

每道题包含：
1. 题目描述（含输入输出示例）
2. 考察点（这道题在测什么）
3. 参考答案
4. 常见错误提示（学生容易踩的坑）

## 约束
- 面向中职学生，避免大学级别的复杂度
- 题目场景贴近生活或学生能理解的情境
- 基础题必须有步骤引导注释
```
