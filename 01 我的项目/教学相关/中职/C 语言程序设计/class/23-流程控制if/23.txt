 第1题：简单判断（⭐）

  编写程序，输入一个整数，如果大于等于0，输出"这是非负数"。       
  // 提示：使用单分支if语句

  第2题：成年判断（⭐）

  编写程序，输入年龄，判断是否成年（≥18岁），输出相应信息。       
  // 提示：使用if-else语句

  第3题：奇偶数判断（⭐⭐）

  编写程序，输入一个整数，判断是奇数还是偶数。
  // 提示：使用取余运算符%

  第4题：最大值比较（⭐⭐）

  编写程序，输入两个数，输出较大的那个数。
  // 提示：考虑相等的情况

  第5题：成绩等级（⭐⭐⭐）

  编写程序，输入成绩（0-100），按以下标准输出等级：
  - 90-100：优秀
  - 80-89：良好
  - 70-79：中等
  - 60-69：及格
  - 0-59：不及格
  // 提示：使用多分支if-else if-else结构

  第6题：三个数的最大值（⭐⭐⭐）

  编写程序，输入三个整数，找出并输出最大值。
  // 提示：使用嵌套if或多个条件判断

  第7题：简单计算器（⭐⭐⭐⭐）

  编写程序实现简单计算器，输入两个数和运算符（+、-、*、/），输出计算结果。注意除零错误处理。
  // 提示：使用字符输入，注意除数为0的情况

  第8题：闰年判断（⭐⭐⭐⭐）

  编写程序判断输入的年份是否为闰年。
  闰年规则：能被4整除但不能被100整除，或者能被400整除。
  // 提示：使用逻辑运算符&&和||

  第9题：字符类型判断（⭐⭐⭐⭐⭐）

  编写程序，输入一个字符，判断它是：
  - 大写字母（A-Z）
  - 小写字母（a-z）
  - 数字字符（0-9）
  - 其他字符
  // 提示：使用ASCII码范围判断

10.输入一个三位数的整数x，判断它是否为水仙花数。水仙花数是指各位数字立方和等于该数，
如：153^3+1^3+5^3+3^3=153。（举一反三，判断是否为阿姆斯特朗数）
