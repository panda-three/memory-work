🌸 水仙花数详解

100~999之间的"魔法数字"

一、什么是水仙花数？
定义：三位数，各位数字的立方和等于该数本身
153 = 1³ + 5³ + 3³
生活类比：
就像拼图游戏：
• 你有一个数字"153"
• 把它拆成3块：1、5、3
• 每块都变成立方体：1³、5³、3³
• 把这3个立方体加起来，神奇地又变回了原数字153！

百位
1³ = 1

5

5³ = 125

3

3³ = 27

153
原数字
验证过程：
① 1³ = 1 × 1 × 1 = 1
② 5³ = 5 × 5 × 5 = 125
③ 3³ = 3 × 3 × 3 = 27
④ 1 + 125 + 27 = 153 ✓ 等于原数字！
二、核心技能：如何拆分数字？
问题：给你一个数字153，怎么用代码得到百位1、十位5、个位3？
方法：除法和取余
个位 = 数字 % 10
十位 = 数字 / 10 % 10
百位 = 数字 / 100 % 10
1
获取个位数字
int num = 153; int ge = num % 10; // 153 % 10 = 3
原理：
153 ÷ 10 = 15 ... 余数3
% 运算符就是取余数，所以 153 % 10 = 3
2
获取十位数字
int shi = num / 10 % 10; // 先除10变成15，再%10得到5
分步执行：
① 先除以10：153 / 10 = 15
② 再取余10：15 % 10 = 5
3
获取百位数字
int bai = num / 100 % 10; // 先除100变成1，再%10得到1
分步执行：
① 先除以100：153 / 100 = 1
② 再取余10：1 % 10 = 1
三、完整代码实现
// 打印所有水仙花数（100~999） #include <stdio.h> int main() { printf("100~999之间的水仙花数有：\n"); // 遍历所有三位数 for (int i = 100; i <= 999; i++) { // 步骤1：分离各个位的数字 int ge = i % 10; // 个位 int shi = i / 10 % 10; // 十位 int bai = i / 100 % 10; // 百位 // 步骤2：计算立方和 int sum = bai*bai*bai + shi*shi*shi + ge*ge*ge; // 步骤3：判断是否等于原数字 if (sum == i) { printf("%d = %d³ + %d³ + %d³\n", i, bai, shi, ge); } } return 0; }
