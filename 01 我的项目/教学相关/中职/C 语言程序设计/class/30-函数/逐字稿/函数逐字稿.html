三、函数的概念

核心定义: 函数就是程序中独立的功能

为什么需要函数?

为了方便大家理解,我们从一个挖掘机的使用工作流程来讲解。

场景一:挖掘机的基本工作流程

挖掘机从启动到熄火,经历这样的流程:

  1. 挖机启动
  2. 挖机开始工作
  3. 挖机工作完成
  4. 挖机熄火
思考问题: 如果我现在上个厕所,或者说挖掘机又有一个新的任务,怎么办呢?

那代码就变成这个样子了:

挖机启动
挖机开始工作
挖机工作完成
挖机熄火

// 上厕所,需要关掉挖掘机

挖机启动
挖机开始工作
挖机工作完成
挖机熄火

// 如果再去上厕所呢?又要重复一遍!
问题发现: 这一段代码在重复地写、重复地写!

四、函数的解决方案

核心思想: 我们可以将这四句代码打包成一个整体,给它起个名字,叫做 goWork

在使用的时候,我只要写这个名字即可!

代码演示

我把这几句话打包,起个名字叫 goWork。注意,这个名字你可以随便起。

// 函数定义 - 把挖掘机从启动到熄火的整个流程打包
void goWork() {
    printf("挖机启动\n");
    printf("挖机开始工作\n");
    printf("挖机工作完成\n");
    printf("挖机熄火\n");
    return 0;
}

// 使用函数
int main() {
    goWork(); // 调用函数
    // 上厕所...
    goWork(); // 再次调用,不需要重复写代码!
    return 0;
}
重点理解:
1. 把某一段代码打包起来,这个就叫做函数
2. 这个功能是独立的,想使用就调用它
3. 函数跟数学上面的函数不太一样!

五、函数的执行流程

代码执行顺序:

  1. 先执行 main 函数里面的第一行代码
  2. 看到 goWork(),是一个函数调用,就会跳到这个函数里面去执行
  3. 在 goWork 函数里面从上往下执行所有语句
  4. 执行完毕后,返回到 main 函数,继续执行下一行代码
  5. 如果又遇到 goWork(),再次跳转执行...

六、函数的分类

1. 系统定义好的函数(库函数)

比如 printf()、scanf() 都是函数!

  • printf - 输出功能,是一个独立的功能
  • scanf - 输入功能,也是一个独立的功能

验证方法: 在代码中按住 Ctrl 键点击函数名,可以看到这个函数的定义!

2. 自定义函数

就是我们刚刚自己写的 goWork,不是系统自带的,是我们自己定义的!

七、重要概念总结

main 也是一个函数!
它是程序的入口函数,所有程序都从 main 函数开始执行。
函数的基本形式:
返回类型 函数名(参数) {
    // 函数体 - 具体的功能代码
    return 返回值;
}
