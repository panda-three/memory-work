三.、程序改错题（共 9 分）
以下程序功能：输入 10 个整数，倒序输出。存在 3 处错误，请指出并改正。
#include <stdio.h>
void main()
{
    int a[10], i, n = 10;
/***** FOUND (1) *****/
    for(i = 1; i <= n; i++)
        scanf("%d", &a[n]);
/***** FOUND (2) *****/
    for(i = n; i >= 0; i--)
        printf("%4d", a[i]);
}
