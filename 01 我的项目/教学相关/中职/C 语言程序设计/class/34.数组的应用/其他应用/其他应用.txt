二、程序改错题（15 分）
1. 程序的功能是将字符串 s 正序和逆序进行连接，形成新的字符串 t。例如，当字符串 s 为 "ABCD" 时，形成的新串 t 为 "ABCDDCBA"。
#include <stdio.h>
void main()
{
    int i, d;
    char s[100], t[100];
    printf("Please enter string s:");
    /******FOUND(1)******/
    scanf("%s", &s);
    d = strlen(s);
    for (i = 0; i < d; i++)
        t[i] = s[i];
    for (i = 0; i < d; i++)
        /******FOUND(2)******/
        t[d + i] = s[d - i];
    /******FOUND(3)******/
    t[2 * d - 1] = '\0';
    printf("The result is: %s\n", t);
}
