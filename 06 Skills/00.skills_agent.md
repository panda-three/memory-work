## Skill Structure

Each skill is a folder containing:
```
skill-name/
├── SKILL.md              ← Main file (YAML metadata + Markdown instructions)
├── scripts/              ← Executable scripts (optional)
├── references/           ← Reference documents (optional)
└── assets/               ← Templates, materials (optional)
```

---

## Skill Types

Skills come in two operational modes:

| Type | Operation | Analogy |
|------|-----------|---------|
| **Protocol** | Written in AGENTS.md, runs automatically every conversation | Subconscious reaction |
| **Skill** | Registered as SKILL.md, loaded when trigger words match | Deliberate invocation |

## Built-in Skills

### Core Skills

| Skill | Purpose | Trigger |
|-------|---------|---------|
| datetime-check | Get calibrated Beijing time | **Mandatory first-load** (auto-executes every session) |
| ics-generator | Generate .ics calendar file from week file | "generate calendar", "export schedule" |
| save-conversation | Save complete conversation as Markdown | "save conversation", "export log" |
| memory-review | Memory system calibration | "week review", "memory review" |
| iteration-log | Log architecture changes | "record iteration", "log change" |

---

## Adding New Skills

You can create new skills anytime:

1. Create a new folder in `06 Skills/`
2. Add `SKILL.md` with metadata and instructions
3. Register trigger words in root `AGENTS.md`
4. AI will automatically recognize and execute
