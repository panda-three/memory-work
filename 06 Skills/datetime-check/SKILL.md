# datetime-check

## 用途

获取校正后的本地时间，为启动序列提供时间上下文。返回结果决定专注区同步档位（轻量 vs 深度）。

## 触发方式

协议型——每次会话启动自动执行，无需用户触发。

## 执行指令

运行以下命令获取当前时间信息：

```bash
date '+%Y-%m-%d %H:%M:%S %A %Z (W%V)'
```

## 输出格式

从命令输出中提取以下字段：

| 字段 | 示例 | 用途 |
|------|------|------|
| 日期 | 2026-02-23 | 进展记录的日期标记 |
| 时间 | 14:30:00 | 会话时间戳 |
| 星期 | Monday | 决定同步档位 |
| 时区 | CST (UTC+8) | 确认时区正确 |
| 周次 | W09 | 匹配 _本周.md 的周次 |

## 下游依赖

- 星期一至三 → 触发轻量同步
- 星期四至日 → 触发深度回溯
- 周次 → 校验 _本周.md 是否为当前周
