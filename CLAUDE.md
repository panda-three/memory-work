# Memory Work - Claude 初始化指令

> 这是一个全新的 Memory Work 项目。当你在这个仓库中被启动时，请遵循以下初始化流程。

---

## 首次启动检测

**每次会话开始时，检查以下条件判断是否需要初始化**：

1. 检查 `USER.md` 是否还是模板状态（包含 `[Your Name]` 或 `[你的名字]`）
2. 检查 `00 Focus Zone/_this_week.md` 或 `00 专注区/_本周.md` 是否存在且有实际内容

**如果是首次使用（USER.md 未配置）**：
→ 执行下方的「首次启动引导」

**如果已配置**：
→ 按 `AGENTS.md` 的正常启动流程执行

---

## 首次启动引导

### Step 1: 语言选择

向用户询问：

```
你好！欢迎使用 Memory Work 🎉

这是一个让 AI 拥有持久记忆的知识管理系统。

首先，请选择你的偏好语言：
1. 中文
2. English

输入 1 或 2：
```

### Step 2: 初始化文件

**如果用户选择「中文」**：

1. 将 `templates/zh-CN/` 下的所有文件复制到对应位置：
   - `templates/zh-CN/AGENTS.md` → 覆盖根目录 `AGENTS.md`
   - `templates/zh-CN/SOUL.md` → 覆盖根目录 `SOUL.md`
   - `templates/zh-CN/USER.md` → 覆盖根目录 `USER.md`
   - `templates/zh-CN/MEMORY.md` → 覆盖根目录 `MEMORY.md`
   - `templates/zh-CN/PROCEDURES.md` → 覆盖根目录 `PROCEDURES.md`
   - `templates/zh-CN/00 专注区/` → 复制为 `00 专注区/`（保留原 `00 Focus Zone/` 或删除）

2. 创建中文目录结构：
   ```
   00 专注区/
   01 素材库/
   02 工具库/
   06 技能库/
   ```

**如果用户选择「English」**：
→ 保持当前文件不变，继续下一步

### Step 3: 用户画像配置

询问用户基本信息：

```
接下来，让我快速了解你，以便更好地协作：

1. 你希望我怎么称呼你？
2. 你的职业/身份是什么？
3. 你目前主要在做什么项目或角色？
4. 有什么专业领域或技能？

你可以简单说，或者口述一段也行，我来整理成档案。
```

根据用户回答，更新 `USER.md`。

### Step 4: 工作区创建

```
好的，[称呼]。让我帮你创建核心文件。

已创建：
- USER.md — 你的档案
- MEMORY.md — 长期记忆库
- [专注区]/_本周.md — 本周工作区（[周数]: [日期范围]）

---

[称呼]，今天[星期几]，新的一周开始。

说说这周想推进什么？或者有什么会议、截止日期、要交付的东西？

你可以随便说，我来帮你整理成任务。
```

### Step 5: Obsidian 引导（如果用户问）

如果用户询问如何在 Obsidian 中打开，回答：

```
在 Obsidian 中打开这个文件夹：

1. 打开 Obsidian
2. 点击左下角的「打开其他仓库」（vault 图标）
3. 选择「打开文件夹作为仓库」
4. 导航到这个路径：

   [当前工作目录的绝对路径]

或者直接在 Finder 里 Cmd + Shift + G，粘贴上面的路径，就能看到这个文件夹的位置。
```

---

## 初始化完成后

完成初始化后，后续会话按正常流程运行：

1. 读取 `SOUL.md` → 了解你是谁
2. 读取 `USER.md` → 了解你在帮助谁
3. 读取 `_本周.md` / `_this_week.md` → 获取当前工作上下文
4. 按需加载 `MEMORY.md` 和 `PROCEDURES.md`
5. 按需搜索全库
